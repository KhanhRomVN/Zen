🖥️ HIỂN THỊ TRÊN UI (React Component):

Khi render lên UI, component sẽ loop qua contentBlocks[] và render từng block:

1. **thinking block** (type: "thinking"):
   - Thường được hiển thị trong 1 collapsible/expandable section
   - Có thể có icon "💭" hoặc "🤔"
   - Styling: màu xám nhạt, font size nhỏ hơn
   - Có thể ẩn/hiện (toggle)

2. **markdown block** (type: "markdown"):
   - Render bằng markdown parser (như react-markdown)
   - Hiển thị bold (**text**), italic, code inline (`code`)
   - Đây là phần CHÍNH mà user thấy
